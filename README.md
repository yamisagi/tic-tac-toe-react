## About The Project

<!-- add asset here later -->
<!--<img src="https://github.com/yamisagi/us/blob/main/assets/product.gif" width="350" height="550"> -->

In this project I built a Tic-Tac-Toe game using ReactJS + Vite. It helped me to understand the importance of the state in React and how to use it. And concepts like immutability, how React handles State and Schedule Updates.
